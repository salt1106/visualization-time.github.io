# スケジュール管理ツール

曜日と時間のグリッド形式でスケジュールを管理できるWebアプリケーションです。

## 機能

- 週間スケジュール表示（日〜土曜日）
- 15分間隔での時間管理
- スケジュールの追加・編集・削除
- ローカルストレージでのデータ保存
- レスポンシブデザイン対応

## ファイル構成

```
schedule-manager/
├── index.html          # メインページ
├── css/
│   └── style.css      # スタイルシート
├── js/
│   └── app.js         # アプリケーションロジック
└── README.md          # このファイル
```

## 使用方法

1. `index.html` をWebブラウザで開く
2. グリッド上の任意のセルをクリックしてスケジュールを追加
3. テキストを入力して確定
4. データは自動的にローカルストレージに保存されます
